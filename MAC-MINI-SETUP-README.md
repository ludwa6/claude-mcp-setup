# Mac Mini Setup Checklist

**Purpose:** Complete setup guide for replicating your Claude Desktop + MCP environment on Mac Mini (Apple Silicon)

**View the complete checklist here:** [MAC-MINI-SETUP.md](MAC-MINI-SETUP.md)

## Quick Start

1. **Install software:** Claude Desktop, VS Code, Node.js
2. **Clone repos:** 
   - `git clone https://github.com/ludwa6/claude-mcp-setup.git`
   - `git clone https://github.com/ludwa6/claude-workspace.git ~/claude-workspace`
3. **Configure MCP:** Copy config to `~/Library/Application Support/Claude/claude_desktop_config.json`
4. **Test connections:** Verify GitHub and WordPress MCP work
5. **Sync workspace:** Pull, make test change, push

See full detailed instructions in MAC-MINI-SETUP.md
